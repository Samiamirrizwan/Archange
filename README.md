> **A sci‑fi, interactive developer portfolio – forged in code, deployed with precision.**  
> This is the personal website of **Sami Amir Rizwan**, a WordPress & Fullstack Developer from Karachi, Pakistan.  
> Inspired by fighter‑jet culture and Ace Combat’s legendary aircraft, the site blends futuristic UI, glassmorphism, live particle systems, and a functional AI terminal.

![ARCHANGE Preview](https://github.com/Samiamirrizwan.png)

---

## ✦ Features

- ▸ **Immersive Sci‑Fi Design** – Glassmorphism cards, 3D parallax tilt, glitch text, holographic avatar, scanlines, and a drifting grid overlay.
- ▸ **Live Interactive Background** – A dynamic particle starfield that responds to mouse movement, simulating deep space.
- ▸ **AI Terminal** – A floating AI assistant that answers questions about skills, experience, projects, and how to hire Sami.
- ▸ **Fully Responsive** – Optimized for all screen sizes from mobile phones to ultrawide monitors.
- ▸ **Professional UI/UX** – Clean typography (Inter, JetBrains Mono, Orbitron), cohesive color theme extracted from the developer’s CV.
- ▸ **No Frameworks** – Pure HTML5, CSS3 (custom properties, flexbox, grid), and vanilla JavaScript. Zero dependencies except web fonts and icons.
- ▸ **Legal Pages** – Includes License, Privacy Policy, and Terms of Service (all with the same sci‑fi styling).
